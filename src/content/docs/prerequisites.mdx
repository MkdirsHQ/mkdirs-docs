## Prerequisites

<Steps>

1. Node.js

   Make sure you have installed Node.js v18 or later, if you don't have it, you can install it from [Node.js official website](https://nodejs.org/en/download/).

   ```bash
   # Check if Node.js is installed
   node -v
   ```

2. Git

   Make sure you have installed Git, if you don't have it, you can install it from [Git official website](https://git-scm.com/downloads).

   ```bash
   # Check if Git is installed
   git --version
   ```

3. PNPM

   Make sure you have installed PNPM, if you don't have it, you can install it from [PNPM official website](https://pnpm.io/installation).

   ```bash
   # Install PNPM
   npm install -g pnpm

   # Check if PNPM is installed
   pnpm --version
   ```

</Steps>
