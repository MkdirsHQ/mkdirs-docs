使用 Mkdirs 模板的前提条件。

## 前提条件

<Steps>

1. Node.js

   请确保你已经安装了 Node.js v18 或更高版本，如果没有，请从 [Node.js 官方网站](https://nodejs.org/en/download/) 安装。

   ```bash
   # 检查 Node.js 是否已安装
   node -v
   ```

2. Git

   请确保你已经安装了 Git，如果没有，请从 [Git 官方网站](https://git-scm.com/downloads) 安装。
   
   ```bash
   # 检查 Git 是否已安装
   git --version
   ```

3. PNPM

   请确保你已经安装了 PNPM，如果没有，请从 [PNPM 官方网站](https://pnpm.io/installation) 安装。

   ```bash
   # 安装 PNPM
   npm install -g pnpm

   # 检查 PNPM 是否已安装
   pnpm --version
   ```

</Steps>
